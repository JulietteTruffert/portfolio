Geospatial 3D data management & web visualisation
Client: NineNorth Inc

I was in charge of the strategy and development of an app to visualize and navigate through the client’s data: massive bathymetric georeferenced point clouds.

This project uses Potreejs library on top of threejs to display large point clouds directly in the web browser.

-- Threejs, Potreejs, HTML5, CSS, Javascript, Python, Geospatial --

Web app creation
Client: Nfinite

This project is a web app to create 3D models quickly and easily without any prior experience in 3D modeling.

It uses parametric strategies to model the objects and adapt the proportions, materials and textures. The concept was successfully applied to artframes and various packaging.

-- Blender, Python, Unity, C#, Parametric 3D modeling, Frontend/Backend development --

Parametric Packaging
Client: Nfinite

This project is a part of a web app to create 3D models of quickly and easily.

It uses advanced modeling features, rigging and animation in Blender in order to get a parametric packaging. This way, the client was able to customize his packaging in size and design without any knowledge in 3D modeling

-- Blender, Python, Rigging, Backend Development --
